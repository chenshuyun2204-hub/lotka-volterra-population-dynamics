# Lotka–Volterra Population Dynamics Simulation

## Overview

This project studies predator–prey population dynamics using the
Lotka–Volterra differential equation model.

The goal of the project is to analyze how interaction parameters
affect population behavior such as extinction, oscillation, and stability.

This research was conducted as a mathematical modeling project at
Coe College under the supervision of Prof. Brittney Miller.

---

## Methods

The Lotka–Volterra model was implemented to simulate nonlinear
predator–prey population dynamics.

Key steps in the project include:

- Numerical simulation of differential equation models
- Parameter sweep experiments (~1,000 parameter settings)
- Generation of more than 10,000 population trajectories

---

## Analysis

Simulation results were analyzed using statistical and machine learning
methods:

- PCA (Principal Component Analysis)
- Hierarchical clustering to classify system behavior
- Logistic regression model to predict population dynamics outcomes

The classification model achieved approximately **0.82 accuracy
using 10-fold cross validation**.
